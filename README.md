# Jump-
A cool nooby platformer game made completely by me!
In ths game,you have to complete 3 levels to win!
Controls are given in game as a tutorial/Controls: WSAD keys for movement,SPACE to jump and DOUBLE SPACE to doubloe jump.
This game was made in Godot engine.

My Journey:
In this project,I learned:
How to make funstions and call them
Change scenes
Make a proper health system
Make a proper enemy AI(basic)
How to design a world in godot
How to find errors in code
How to export a game and put in itch
